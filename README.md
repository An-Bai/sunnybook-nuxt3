# suunybook-nuxt3
### 开发说明
个人全栈开发博客类型项目sunnybook之前端部分

<br/>


### 前端项目开发技术选型
Nuxt3 + typescript + scss + markdown-it&highlight.js
+ 简要说明：Nuxt3是基于 vue3 的开源框架，除了SSO 方面的提升，还有很多有趣方便的功能。Nuxt3在2022年10月26日发布正式版本。


<br/>


### 关于
第一个版本是v1版本，还只是一个非常简陋的前端模板框架，第二个版本会结合后端使用。同时，我也会更新后端项目。
+ 除基础框架Nuxt3外，我决定不使用大型的如element-ui等js框架，页面交互效果几乎会由自己实现，因为我喜欢设计，我会把我的想法试验在我的博客项目js互动和css美化中。
+ 为什么不用WordPress、Hogo之类的CMS搭建？哦~，我了解过也非常喜欢他们的设计和主题，但是我更喜欢自己手动设计，一位这个博客也许会陪我很久，我也有时间就会美化它。而且最近对“小土拨鼠”很感兴趣，它真是太可爱啦！所以迫不及待地准备用我对它为数不多的了解和巨大的热情做点什么。写一个自己的博客是一个非常不错的选择。


### 使用
v1 是纯前端基础框架，只要有node.js（本人使用的版本是18.8.0）环境就可以运行，启动如下：

+ 初始化项目

```powershell
npm install
```

+ 运行启动项目

```powershell
npm run dev
```
